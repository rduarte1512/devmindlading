FAQ

Perguntas frequentes.

O que é o preçário por API Keys?

É uma forma de cobrar pela utilização dos modelos DevMind através de chaves de API, normalmente por tokens, por volume incluído e por limites de utilização.

Posso ter várias API Keys?

Sim. Cada plano pode limitar o número de chaves. Isto permite separar projetos, ambientes de teste, produção e clientes.

Os preços são finais?

Não. Estes valores são uma proposta comercial fictícia para a landing page. Devem ser ajustados quando houver custos reais de infraestrutura e modelo.
